Разработка платежного компонента для Joomshopping.

Сентябрь 23, 2014 pronata

Под Joomla CMS имеются, как минимум, два бесплатных решения для создания интернет-магазина. Это VirtueMart и Joomshopping, про создание расширения под второе решение речь и пойдет в моем первом посте.

Передо мной была поставлена задача создания плагинов под CMS для упрощенной интеграции с платежным шлюзом. Дело дошло до Joomshopping. Для создания и тестирования плагина потребовалось установить Joomshopping на Joomla. Воспользовалась ссылкой для скачивания Joomshopping и рекомендациями по установке оттуда же.

Приступаем к созданию компонента.

В установочном архиве создаваемого платежного компонента обязательно будет следующая вложенность, повторяющая в точности вложенность в 

joomshopping components\com_jshopping\payments\

Sub-heading

Имеется класс payments.php, в котором лежат методы, которые потребуется переопределять под наши нужды.
